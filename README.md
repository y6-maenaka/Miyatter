<h1>Miyatter</h1>
Miyatterは、Django REST Framework（DRF）とDjango Channelsを利用して開発した、Twitterと2chを融合させたようなリアルタイムWebコミュニケーションアプリです。<br>
投稿内容は全て非同期で接続しているユーザに即時配信され、最新の情報がリアルタイムに反映されます。
